Limitations
--------------------

Some widgets should be used only in Page Builder Grid: BootFrame Section Header, BootFrame Testimonials, BootFrame Portfolio Items, BootFrame Our Team Columns, BootFrame Display Page Content, BootFrame Counter Boxes, BootFrame Call to Action
Custom background color or image is displayed at a resolution higher than 1280 px
